# 🎓 Student Management System (MERN Stack)

A full-stack **Student Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This application allows users to efficiently manage student records with complete CRUD operations.

---

## 🚀 Features

✨ **Student Management**
- ➕ Add new students
- 📋 View all students
- ✏️ Update student details
- 🗑️ Delete student records

🔍 **Search & Management**
- Search students easily
- View student information
- Manage student data efficiently

🛡️ **Validation & Error Handling**
- Basic form validation
- API error handling
- Secure data management
